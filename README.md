# MCT231-BASICS-OF-MECHATRONICS
MCT231 Mechatronics Project: A fully integrated, standalone system featuring sorting, lifting, and closed-loop projectile mechanisms. Built with a custom PCB, strict cable management, and robust safety standards. It includes Bluetooth PC comms, an on-board LCD, and a visual Processing GUI. Repo contains all Code, CAD, and Docs.
# PlanetSort: OrbitQuest 🌌

An autonomous, space-themed interactive mechatronic system designed as an educational STEM toy. The project features automated color-based planet sorting, continuous vertical elevator lifting, and a calibrated dual-wheel friction projectile launcher. 

Developed as the Final Project for the **Basics of Mechatronics (MCT231)** course under the partnership of **Ain Shams University** and the **University of East London**.

## 🚀 Project Architecture Overview
**PlanetSort: OrbitQuest** is engineered as a fully closed-loop, self-contained system running within a strict 100 cm x 100 cm x 100 cm volume. The product gamifies automation principles by treating different colored marbles as distinct planets navigating an interstellar transit grid. 

The system runs entirely off a single unified **12V DC power supply** and delegates tasks across a low-latency, wireless **Dual-ESP32 Master-Slave network** using the ESP-NOW protocol.
